# Введение в MASS Handbook

Добро пожаловать в **Справочник для разработчиков автономных морских судов (MASS Handbook)**!

Этот ресурс создан специально для инженеров, исследователей и разработчиков, работающих над технологиями автономного судовождения. Здесь собраны передовые научные статьи, практические гайды и глоссарий терминов, которые помогут вам в проектировании надежных и безопасных систем для морских автономных надводных судов (Maritime Autonomous Surface Ships).

## Что вы найдете в справочнике?

1. **[База научных статей](/docs/papers)**: Автоматически пополняемая коллекция релевантных публикаций из мировых баз данных (CrossRef, arXiv, IEEE). Все статьи аннотированы и содержат ключевые выводы для практического применения.
2. **[Практические гайды](/docs/guides)**: Руководства по реализации алгоритмов навигации, систем предотвращения столкновений и интеграции с бортовыми сенсорами.
3. **[Глоссарий терминов](/docs/glossary)**: Единый словарь специфических терминов, аббревиатур и понятий из области MASS и морской навигации.

## Архитектура системы

Справочник не только предоставляет доступ к знаниям, но и демонстрирует передовые подходы к автоматизации:

- **Автоматический сбор**: Скрипты регулярно опрашивают научные базы данных для поиска новых публикаций по ключевым словам (например, `collision avoidance`, `DWA`, `knowledge maps`).
- **Интеллектуальная обработка**: Используются LLM (большие языковые модели) для перевода аннотаций, выделения сути и оценки уровня готовности технологии (TRL).
- **Непрерывная интеграция**: Справочник построен на базе Docusaurus и автоматически развертывается через CI/CD пайплайн.

## Как использовать справочник

Используйте боковое меню для навигации по разделам или воспользуйтесь функцией поиска (будет доступна в полной версии) для быстрого нахождения нужной информации по ключевым словам.

> "Разработка автономных судов — это не просто автоматизация существующих процессов, это переосмысление всей парадигмы морской навигации с фокусом на безопасность и эффективность."
