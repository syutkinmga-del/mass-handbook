# Dynamic Tabu Search for Collision Avoidance in Autonomous Maritime Ships

**Авторы:** Burak Alptekin, Nihan Kahraman
**Год:** N/A
**Источник:** crossref
**DOI:** 10.1109/access.2024.3419706
**arXiv ID:** N/A

## Аннотация (оригинал)

Collision avoidance remains a critical challenge in the deployment of autonomous maritime ships, where dynamic and uncertain environments demand robust and efficient path-planning algorithms. This paper presents a novel Dynamic Tabu Search (DTS) approach tailored for real-time collision avoidance in autonomous vessels navigating congested waterways. The proposed methodology integrates a dynamic neighborhood structure with adaptive tabu tenure, enabling efficient exploration of the solution space while preventing cycling and premature convergence. By incorporating real-time sensor data and predictive models of surrounding vessel trajectories, the DTS algorithm dynamically updates its search parameters to respond to evolving maritime scenarios. Key contributions include the development of an enhanced tabu search framework optimized for continuous maritime navigation, and a comprehensive evaluation against benchmark collision avoidance algorithms under varied traffic densities and environmental conditions. Simulation results demonstrate that the DTS approach significantly improves collision avoidance success rates, reduces path deviation, and enhances computational efficiency compared to conventional methods. The findings suggest that Dynamic Tabu Search offers a viable and effective solution for autonomous maritime navigation systems, contributing to safer and more reliable ship operations in complex marine environments.

## Аннотация (русский перевод)

Избежание столкновений остается критической задачей при внедрении автономных морских судов, где динамичные и неопределенные условия требуют надежных и эффективных алгоритмов планирования маршрута. В данной работе представлен новый подход на основе динамического табу-поиска (Dynamic Tabu Search, DTS), адаптированный для предотвращения столкновений в реальном времени у автономных судов, курсирующих по загруженным водным путям. Предложенная методология интегрирует динамическую структуру соседства с адаптивной длительностью табу, что обеспечивает эффективный поиск в пространстве решений при предотвращении цикличности и преждевременной сходимости. За счет использования данных с датчиков в реальном времени и предиктивных моделей траекторий окружающих судов алгоритм DTS динамически обновляет параметры поиска для адаптации к изменяющимся морским условиям. Ключевые результаты включают разработку усовершенствованной структуры табу-поиска, оптимизированной для непрерывной морской навигации, а также комплексную оценку по сравнению с эталонными алгоритмами предотвращения столкновений при различных плотностях трафика и условиях окружающей среды. Результаты моделирования демонстрируют, что подход DTS значительно повышает эффективность предотвращения столкновений, снижает отклонения маршрута и улучшает вычислительную производительность по сравнению с традиционными методами. Полученные данные свидетельствуют о том, что динамический табу-поиск представляет собой жизнеспособное и эффективное решение для систем автономной морской навигации, способствуя более безопасной и надежной эксплуатации судов в сложных морских условиях.

## Ключевые выводы

- Introduced a novel Dynamic Tabu Search (DTS) algorithm with dynamic neighborhood structure and adaptive tabu tenure for real-time collision avoidance in autonomous maritime vessels.
- Integrated real-time sensor data and predictive models of surrounding vessel trajectories to dynamically update search parameters in response to changing maritime scenarios.
- Developed an enhanced tabu search framework optimized for continuous navigation in congested waterways, preventing cycling and premature convergence.
- Demonstrated through simulations that DTS significantly improves collision avoidance success rates, reduces path deviation, and enhances computational efficiency compared to benchmark algorithms.
- Validated the DTS approach as a viable and effective method for safer and more reliable autonomous ship operations in complex and dynamic marine environments.
