# A Dynamic Programming Approach to the Collision Avoidance of Autonomous Ships

**Авторы:** Raphael Zaccone
**Год:** 2024
**Источник:** crossref
**DOI:** 10.3390/math12101546
**arXiv ID:** N/A

## Аннотация (оригинал)

The advancement of autonomous capabilities in maritime navigation has gained significant attention, with a trajectory moving from decision support systems to full autonomy. This push towards autonomy has led to extensive research focusing on collision avoidance, a critical aspect of safe navigation. Among the various possible approaches, dynamic programming is a promising tool for optimizing collision avoidance maneuvers. This paper presents a DP formulation for the collision avoidance of autonomous vessels. We set up the problem framework, formulate it as a multi-stage decision process, define cost functions and constraints focusing on the actual requirements a marine maneuver must comply with, and propose a solution algorithm leveraging parallel computing. Additionally, we present a greedy approximation to reduce algorithm complexity. We put the proposed algorithms to the test in realistic navigation scenarios and also develop an extensive test on a large set of randomly generated scenarios, comparing them with the RRT* algorithm using performance metrics proposed in the literature. The results show the potential benefits of an autonomous navigation or decision support framework.

## Аннотация (русский перевод)

Развитие автономных возможностей в морской навигации привлекло значительное внимание, с тенденцией перехода от систем поддержки принятия решений к полной автономии. Этот переход к автономии вызвал обширные исследования, сосредоточенные на предотвращении столкновений — критически важном аспекте безопасной навигации. Среди различных возможных подходов динамическое программирование является перспективным инструментом для оптимизации маневров по предотвращению столкновений. В данной работе представлена формулировка задачи динамического программирования для предотвращения столкновений автономных судов. Мы формируем рамки задачи, формулируем её как многоступенчатый процесс принятия решений, определяем функции затрат и ограничения, ориентируясь на реальные требования, которым должен соответствовать морской маневр, и предлагаем алгоритм решения с использованием параллельных вычислений. Кроме того, представлен жадный аппроксимационный метод для снижения сложности алгоритма. Предложенные алгоритмы были протестированы в реалистичных навигационных сценариях, а также проведено обширное тестирование на большом наборе случайно сгенерированных сценариев с последующим сравнением с алгоритмом RRT* с использованием метрик производительности, предложенных в литературе. Результаты демонстрируют потенциальные преимущества автономной навигационной системы или системы поддержки принятия решений.

## Ключевые выводы

- The paper formulates collision avoidance for autonomous vessels as a multi-stage dynamic programming (DP) problem with cost functions and constraints aligned with marine maneuver requirements.  
- A solution algorithm leveraging parallel computing is proposed to efficiently solve the DP formulation.  
- A greedy approximation method is introduced to reduce the computational complexity of the algorithm.  
- The proposed algorithms are tested in realistic and extensive randomly generated navigation scenarios.
